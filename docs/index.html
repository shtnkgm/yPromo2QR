yPromo2QR / プロモコード簡単入力 


                        

                        


                    

                    
設定

                    
                    

                        Markdown記法のTodoプレフィックス（- [ ]）を付与することで、

                        チェックの有無により、プロモコードの利用有無を管理できます。

                        
出力例
 - [ ] ABC1234XXX([QRコード](https://QRコードリンク))
 - [ ] DEF5678XXX([QRコード](https://QRコードリンク))
